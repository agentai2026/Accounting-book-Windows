## 根目录精简视图（日常只看这些）

```
记账本电脑端/
├── 启动.bat          ⭐ 双击启动
├── pubspec.yaml      项目配置（依赖）
├── README.md         简要说明
├── lib/              ⭐ 全部源代码
├── assets/           图标资源
├── docs/             文档
├── test/             测试
├── tool/             脚本工具
├── windows/          Windows 打包（Flutter 必需，勿删）
├── macos/            macOS 打包
└── linux/            Linux 打包
```

### 已隐藏 / 可忽略的项

| 项 | 说明 |
|----|------|
| `.dart_tool/` `build/` | Flutter 编译缓存，运行 `tool/tidy_root.bat` 可在资源管理器中隐藏 |
| `.metadata` `.flutter-plugins-dependencies` | Flutter 自动生成 |
| `.cursor/` | Cursor AI 规则（由原 `.cursorrules` 迁入） |
| `.vscode/` | 编辑器配置（折叠显示缓存目录） |
| `.idea/` `*.iml` | 已删除，IDE 需要时会自动重建 |

**资源管理器变清爽**：双击 `tool/tidy_root.bat`  
**恢复显示缓存**：双击 `tool/show_dev_folders.bat`

## 根目录完整说明

```
记账本电脑端/
├── lib/              ⭐ 全部源代码（只看这个写功能）
├── assets/           图标、图片、字体资源
├── docs/             文档、截图
├── test/             单元测试
├── tool/             开发脚本（初始化、启动、工具）
│
├── windows/          Flutter 平台配置（Windows 打包，勿删）
├── macos/            Flutter 平台配置（macOS 打包，勿删）
├── linux/            Flutter 平台配置（Linux 打包，勿删）
│
├── build/            编译产物（可删，flutter clean 会清）
└── .dart_tool/       Flutter 工具缓存（自动生成，勿提交）
```

### 哪些不能删？

| 目录 | 原因 |
|------|------|
| `lib/` | 业务代码 |
| `assets/` | pubspec 引用的资源 |
| `windows/` `macos/` `linux/` | Flutter 桌面端必需 |
| `pubspec.yaml` | 依赖与项目配置 |

### 哪些可以忽略？

| 目录/文件 | 说明 |
|-----------|------|
| `build/` | 编译输出，已在 `.gitignore` |
| `.dart_tool/` | 自动生成 |
| `.idea/` | IDE 配置，已在 `.gitignore` |

## lib/ 源码结构（两层架构）

```
lib/
├── main.dart           程序入口
├── app.dart            MaterialApp、主题
├── routes.dart         路由表
│
├── core/               核心层（以后手机端也能用）
│   ├── models/         数据模型（17 个）
│   ├── database/       SQLite + DAO + 迁移
│   ├── services/       业务逻辑（约 30 个）
│   ├── rules/          ⭐ 可配置规则（AI识图 / 账单导入 / 统计口径）
│   ├── constants/      常量（旧路径 re-export，兼容用）
│   └── utils/          工具函数
│
└── desktop/            桌面端 UI
    ├── pages/          15 个主页面
    ├── widgets/        UI 组件（见下方）
    ├── providers/      Riverpod 状态（19 个）
    ├── services/       窗口/托盘/快捷键/OCR
    ├── theme/          主题颜色
    └── constants/      桌面端专用常量
```

## tool/ — 开发脚本

| 脚本 | 用途 |
|------|------|
| `setup.ps1` / `setup.bat` | 首次初始化（flutter create、pub get） |
| `run.ps1` / `run.bat` | 启动 Windows 开发版 |
| `tidy_root.bat` | 隐藏根目录编译缓存（资源管理器更清爽） |
| `show_dev_folders.bat` | 恢复显示被隐藏的缓存目录 |
| `generate_icons.ps1` | 从 `assets/icons/app_icon.png` 生成 exe/托盘图标 |
| `创建桌面快捷方式.bat` | 在桌面创建带图标的「轻记账」快捷方式 |
| `analyze_alipay_csv.dart` | 支付宝 CSV 分析工具 |
